## IDE

### 是什么？在哪用？

- **作用**：为开发者提供的集成的开发环境，包括了各种编辑功能、管理功能和辅助工具

### 功能

### 是什么？在哪用？

- **工程管理**
  - **管理工程的代码和资源**
  - **管理工程之间或者代码的版本和依赖**
  - **管理编译相关的配置**

- **编辑器**
  - **便捷高效的代码编辑**
  - **代码提醒**

- **编译器**：编译代码

- **调试器**：调试程序

- **其它各种工具**
  - **性能**：性能分析工具
  - **日志**：日志查看工具
  - **代码分析**：在编译之前去发现各种潜在的问题
  - **代码重构**：重构代码
  - **单元测试**：运行单元测试

### 候选

### 是什么？在哪用？

- **这些IDE都支持多种编程语言**

- **Visual Studio**
  - **应用举例**：可用于Unity客户端（C#）开发与调试

- **Web Storm**
  - **应用举例**：可用于后端开发与调试，比如nodejs

- **IntelliJ IDEA**
  - **应用举例**：可用于Lua（Unity的热更集成）的开发与调试

- **JetBrains Rider**
  - **应用举例**：常被用于开发调试Unity（C#）、UE（C++）的脚本或代码

### 会遇到哪些问题？

- **IDE选择**
  - **问题**：不同IDE有不同的特点和适用场景
  - **解决方向**：根据项目需求、编程语言、团队习惯等因素选择合适的IDE

## 版本控制工具

### 是什么？在哪用？

### Perforce

- **说明**：一款非常强大的版本控制系统，适合于大型项目，可以很有效的管理大量大尺寸文件
- **特点**：收费系统
- **适用于**：
  - **大的工程**
    - 团队大
    - 资源多
    - 文件大

### Git

- **说明**：分布式版本控制系统
- **特点**：开源免费
- **候选客户端**：
  - **Tortoise Git**：Windows上非常好用的Git客户端
  - **SourceTree**：跨平台的Git客户端，也支持Mac系统

### SVN

- **说明**：中心式版本控制系统
- **特点**：开源免费
- **候选客户端**：
  - **Tortoise SVN**：Windows上的SVN客户端

### 会遇到哪些问题？

- **版本控制工具选择**
  - **问题**：不同工具有不同的特点和适用场景
  - **解决方向**：
    - 大型项目、大文件多的情况可以考虑Perforce
    - 中小型项目、开源项目通常使用Git
    - 传统企业项目可能使用SVN

## 比较工具

### 是什么？在哪用？

- **作用**：比较不同版本的文件、目录的差别

### 应用举例

### 是什么？在哪用？

- **版本管理**
  - **说明**：这是最常用的场景，每次提交代码或数据的时候，查看一下文件的变化，确保提交的内容都是确定需要提交的

- **查问题**
  - **说明**：一个问题在某个版本之后开始出现了，可以比较两个提交版本之间发生了哪些变化

- **同步文件**
  - **说明**：有两处代码或数据，一处用于开发编辑，一处用于运行调试，但改动一处后，两台机器或两个目录的内容就不一致了。如果需要同步，可以借助比较工具

### 候选工具

### 是什么？在哪用？

- **Beyond Compare**：功能强大的文件和目录比较工具

- **Win Merge**：开源的Windows文件和目录比较工具

## 代码分析工具

### 是什么？在哪用？

- **作用**：帮助开发者检查代码中潜在的问题

### 举例

- **类型检查**：检查类型错误
- **赋值顺序检查**：检查赋值顺序问题
- **格式检查**：检查代码格式

### 候选工具

### 是什么？在哪用？

- **PC-Lint**
  - **说明**：最知名的C++的代码静态分析工具

- **FxCop**
  - **说明**：C#的代码静态分析工具

- **PVS-Studio**
  - **说明**：PVS‑Studio 是一种用于检测 Windows、Linux、macOS 上的 C、C++、C# 和 Java 源代码中的错误和潜在漏洞的工具

### 会遇到哪些问题？

- **代码分析工具选择**
  - **问题**：不同工具支持不同的编程语言和平台
  - **解决方向**：根据项目使用的编程语言和平台选择合适的代码分析工具
